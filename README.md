Data analysis and visualization project that explores Superstore sales data to identify trends, top-performing categories, and regional performance using Python.
# Sales Dashboard Project

##  Objective
Analyze Superstore sales data and create a dashboard to extract useful business insights.

##  Tools Used
- Python
- Pandas
- Matplotlib
## Dataset
- Superstore Sales Dataset(CSV)

## Features
- Sales by Category (Bar Chart)
- Monthly Sales Trend (Line Chart)
- Region-wise Sales (Pie Chart)

##  Insights
- Identified top-performing categories
- Analyzed monthly sales trends
- Compared sales across regions

##  Files
- sales_dashboard.ipynb
- Sample - Superstore.csv

## 🚀 Conclusion
This project demonstrates basic data analysis and visualization skills using Python.
